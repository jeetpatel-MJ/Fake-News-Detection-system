# Fake-News-Detection-system
Developed a Fake News Detection System using Machine Learning and NLP. Implemented TF-IDF vectorization with Logistic Regression, SVM, Random Forest, and Naïve Bayes classifiers. Features a clean web interface for real-time prediction with ~94% accuracy. Aimed at fighting misinformation.


Fake News Detection System
A Machine Learning based web application that detects fake news articles using Natural Language Processing (NLP).
Overview
Developed an intelligent system to classify news articles as Real or Fake. The model uses TF-IDF vectorization and multiple classification algorithms to achieve high accuracy in identifying misinformation.
Features

Real-time fake news detection
User-friendly web interface
Multiple ML models comparison
High accuracy (~94%)
Supports text input and URL (future scope)

Technologies Used

Python
Machine Learning: Scikit-learn
NLP: TF-IDF Vectorizer
Models: Logistic Regression, SVM, Random Forest, Naïve Bayes
Frontend: HTML, CSS, JavaScript
Backend: Flask / Streamlit (as per your project)

Dataset

Used publicly available Fake and Real News datasets (Kaggle)

Results

Best Model Accuracy: ~94%
Successfully combats the spread of misinformation
